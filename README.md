# 🏠 House Haunter

**House Haunter** is a modern landing page implementation for a real estate search platform. This project was built to demonstrate proficiency in creating clean, and visually appealing user interfaces using core web technologies.

> [!CAUTION]
> **Disclaimer:** The UI design of this project does not belong to me. It has been used solely for educational and portfolio purposes to demonstrate my front-end development skills. This design should not be used for commercial purposes.

---

## 🌟 Overview

The goal of this project was to implement a high-fidelity, interactive front page that helps users find their perfect home. 

While it currently serves as a static landing page, it is built with scalability and clean architecture in mind.

### Key Features:
* **Fully Responsive:** Optimized for a seamless experience across mobile, tablet, and desktop devices.
* **Modern Styling:** Utilizes advanced SCSS for modular and maintainable styles.
* **High Performance:** Powered by Vite for lightning-fast development and optimized asset loading.

---

## 🛠️ Tech Stack

This project was built without heavy frameworks to showcase mastery of the fundamentals:

* **HTML5** – Semantic structure for better SEO and accessibility.
* **SCSS / CSS** – Advanced styling using variables, nesting, and flexible layouts (Flexbox/Grid).
* **Vanilla JavaScript** – Interactive UI elements and logic.
* **Vite** – Modern build tool and development server.

## 🚀 Getting Started

To get this project up and running locally, ensure you have **Node.js** installed. Then, follow these steps in your terminal:

```bash
# 1. Clone the repository
git clone [https://github.com/your-username/house-haunter.git](https://github.com/your-username/house-haunter.git)

# 2. Navigate into the project folder
cd house-haunter

# 3. Install dependencies
npm install

# 4. Launch the development server
npm run vite
